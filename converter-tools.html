Frequently Asked Questions

What Word document elements are preserved when converting to PDF?

Headings H1 through H6, bold, italic, underlined text, ordered and unordered lists, tables with borders, hyperlinks, and embedded images are all preserved. Custom fonts are substituted with browser-safe equivalents. Complex multi-column layouts, text boxes, and advanced header and footer designs may not carry through exactly — for pixel-perfect output use Microsoft Word's own Export to PDF function.

What Markdown syntax does the converter support?

CommonMark specification plus GitHub Flavored Markdown extensions including tables, task lists, and strikethrough. Code blocks with language identifiers produce the correct HTML class attributes for syntax highlighting libraries such as Prism.js. Footnotes and LaTeX mathematical notation are not currently supported.

Why use WebP instead of GIF for animations?

Animated WebP uses modern compression algorithms that reduce file sizes by 50–70% compared to GIF, while supporting a full colour palette rather than GIF's 256-colour limit per frame. Most modern browsers and devices support animated WebP. GIF remains the safer choice for email clients and older platforms that have not yet added WebP support.

What favicon sizes should I generate?

The minimum set for modern web use: 16×16 for the browser tab, 32×32 for the bookmarks bar, 180×180 for the Apple Touch Icon on iOS home screens, 192×192 for Android Chrome, and 512×512 for Progressive Web Apps. The Favicon Generator outputs all five plus a 48×48 icon for Windows. Start with a square, high-contrast source image at 512×512 or larger for best results.

Can the Image Collage Maker handle photos with different aspect ratios?

Yes. The collage maker supports 2, 3, and 4-cell grid layouts. Each cell crops and centres its source image to fill the cell proportions automatically. You can adjust the gap between cells and download the final collage as JPG or PNG at a resolution you specify.
