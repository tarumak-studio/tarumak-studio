Frequently Asked Questions

What image formats are supported?

JPG, PNG, WebP, SVG, and GIF are all supported across the image tool collection. Individual tools specify which formats they accept — the Image Compressor handles JPG, PNG, and WebP, while the SVG to PNG Converter specifically takes SVG input and produces a raster PNG output at a resolution you choose.

Are there any file size limits?

There are no server-imposed limits because files never leave your device. Practical limits depend on your browser's available memory — most modern browsers handle images up to 50MB without issues. Very large RAW camera files may be slow to process depending on your device.

Will image quality be affected by compression?

Lossless operations such as resizing, cropping, and format conversion to PNG preserve full quality. Lossy operations such as JPEG compression and conversion to JPG or WebP let you control the quality level with a slider. The Image Compressor shows the file size reduction percentage in real time as you adjust quality.

Do the tools work on iPhone and Android?

Yes. All image tools use standard browser APIs supported by Safari on iOS 14 and later, Chrome for Android, and all modern mobile browsers. File picker, canvas processing, and file download all work correctly on mobile devices.

Can I process multiple images at once?

The Image Resizer and Image Compressor support batch processing — upload multiple files and download them individually or as a ZIP archive. Single-operation tools like the Image Cropper process one image at a time.
