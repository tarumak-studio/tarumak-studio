HOW PRIVACY WORKS

Nothing to trust because nothing to send

Every tool on Tarumak Studio runs the same way — here is exactly what that means in practice.

Runs on your device

Your browser does the work using APIs like Canvas and FileReader — the same engine as any other tab, just pointed at your files instead of a website.

Nothing is uploaded

File tools process locally and never leave your device; text and URL tools process what you type or paste the same way. Either way, there is no upload step to intercept.

Built on open libraries

PDF tools use pdf-lib and PDF.js, OCR uses Tesseract.js, Word conversion uses mammoth.js — established open-source projects, not a black box.

No account, no tracking your files

No sign-up gate sits in front of any tool. Basic, cookieless-by-default analytics measure traffic — never the content of what you process.
